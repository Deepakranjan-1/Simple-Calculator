Simple Calculator
An interactive and visually appealing calculator built using HTML, CSS, and JavaScript.
It offers a sleek user interface and supports both mouse clicks and keyboard operations for convenience.

Highlights:
- Perform basic operations: Add, Subtract, Multiply, Divide
- Supports decimals, percentages, clear (C), and backspace (DEL)
- Fully responsive layout with a modern and creative look
- Keyboard-friendly for quick input

How to Use:
Download or clone this repository.

Open the index.html file in any web browser to start using the calculator.
