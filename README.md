# Professional Calculator

A sleek, modern calculator with a professional black and white theme, featuring smooth animations and premium visual effects. Built with HTML, CSS, and JavaScript for an exceptional user experience.

## Features

### Design & Aesthetics
- **Premium Black & White Theme**: Sophisticated monochrome design with high contrast
- **Animated Background**: Dynamic grid pattern that moves subtly in the background
- **Glassmorphic UI**: Modern calculator design with gradient shadows and depth
- **Professional Branding**: Clean header with "CALC." logo and status indicator
- **Dual-Line Display**: Shows both current input and previous calculation

### Functionality
- **Basic Operations**: Addition, Subtraction, Multiplication, Division
- **Advanced Features**: Percentage calculations, decimal support
- **Smart Input**: Auto-adds leading zero for decimals, prevents invalid operations
- **Clear & Delete**: Full clear (C) and backspace (DEL) with icon
- **Result History**: Secondary display shows your last calculation

### Animations & Effects
- **Button Interactions**: Ripple effect on press, hover elevation, smooth transitions
- **Display Animations**: Glow effect when numbers update
- **Page Load**: Smooth fade-in and scale animation
- **Visual Feedback**: Pulsing status indicator, shimmer effects
- **Responsive Design**: Adapts beautifully to all screen sizes

### User Experience
- **Keyboard Support**: Full keyboard input for faster calculations
  - Numbers and operators: `0-9`, `+`, `-`, `*`, `/`, `%`, `.`
  - Enter or `=`: Calculate result
  - Backspace: Delete last character
  - Escape: Clear all
- **Touch Optimized**: Perfect for mobile and tablet devices
- **Accessible**: High contrast design for better readability

## How to Use

1. Download or clone this repository
2. Open `index.html` in any modern web browser
3. Start calculating with mouse clicks or keyboard input
